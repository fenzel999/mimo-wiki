# index.md 模板

索引按类型分区。每个条目一行：wikilink + 摘要。

```markdown
# Wiki Index

> 内容目录。每个 wiki 页面列在其类型下，附带一行摘要。
> 先读这个来找到任何查询的相关页面。
> 最后更新: YYYY-MM-DD | 总页面数: N

## 实体
<!-- 分区内按字母顺序 -->

## 概念

## 比较

## 总览

## 查询
```

**扩展规则：** 当任何分区超过 50 个条目时，按首字母或子领域拆分为子分区。
当索引超过 200 个条目时，创建 `_meta/topic-map.md` 按主题分组页面以便更快导航。
